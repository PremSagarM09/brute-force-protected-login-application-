Steps to Run the Project Locally
- Option 1: Using Individual READMEs

Follow the README files in the frontend and backend folders to run each part separately.

- Option 2: Using Commands
    - Step 1: Start Frontend
    - Open one terminal in the project root:
    - `npm run frontend-build`   # Build the frontend
    - `npm run frontend-start`   # Start the frontend server
    
    - Step 2: Start Backend
    - Open another terminal in the project root:
    - `npm run backend-build`    # Build the backend
    - `npm run backend-start`    # Start the backend server
