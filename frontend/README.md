# Login App (Vite + React)

Simple demo app with:
- Login page using Material UI
- Email and password validation (password min 8 chars, valid email)
- Snackbar for success and error messages
- Welcome page showing user name on successful login

## Run locally
1. npm install
2. npm run dev
